## What is included?

As of today, the following networks have been implemented:

- [x] DQN [[1]](http://arxiv.org/abs/1312.5602)
- [x] DDPG [[2]](http://arxiv.org/abs/1509.02971)
- [x] TD3
- [ ] PPO
